StackLens Privacy Policy

Effective date: 2026-07-11

Summary

StackLens is a local browser utility that inspects public Shopify storefront signals from the active tab and displays results in a local dashboard. It does not send personal data, browsing activity or scan results to a developer-operated server.

What data and content is processed

The extension processes the active storefront URL, public DOM and page-state data, theme hints, app surfaces, pixel signals, SEO and performance signals, and public storefront endpoint responses. It does not access Shopify Admin, orders, customers, payments, revenue or private analytics.

How the data is used

This information is used only to analyze the active public storefront and present the resulting signals in the extension's local dashboard.

Sharing and transmission

The extension does not transmit storefront results or extension state to the developer. To perform a scan, the user's browser may directly request public pages and public storefront endpoints from the active target domain. Opening an app listing is performed only when the user chooses the relevant link.

Chrome storage

The current scan target, popup-window state and local dashboard state are stored in Chrome extension storage. This data is not sent to the extension developer and is not described as Chrome sync data. It remains locally until replaced, cleared by the user or removed when the extension is uninstalled.

Data sale and prohibited uses

The extension does not sell user data, use it for advertising, or use it for creditworthiness or lending purposes.

Remote code

The extension does not load or execute remotely hosted code. All executable JavaScript is included in the installed extension package.

Limitations

StackLens inspects public storefront signals only. It does not bypass access controls, Shopify Admin boundaries or private analytics. Results are best-effort observations of signals visible to the browser and may not identify every technology or configuration.

Retention and deletion

The developer does not retain extension data because no data is sent to a developer-operated server. Users can remove local extension state by clearing the extension's data or uninstalling it.

Contact

For support or privacy questions, use the support URL on the extension's Chrome Web Store item page.
