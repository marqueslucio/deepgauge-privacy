DeepGauge - Website Optimizer

Privacy Policy

DeepGauge is a local browser extension for scanning active-page performance, resource weight, code-health and page-type optimization-risk signals after the user clicks the extension.

Effective Date

June 24, 2026

Summary

DeepGauge processes scan data locally in the user's browser. It does not send scan results, page content, exported reports or extension settings to a developer-operated server. It does not sell user data, use user data for advertising, or use user data for creditworthiness or lending purposes.

What Data DeepGauge Processes

When the user clicks the extension and runs a scan, DeepGauge may process browser-visible information from the active tab to provide the audit dashboard:

  • Page URL and page title.
  • Navigation timing, resource timing and visible transfer-size signals when exposed by the browser.
  • DOM-health and code-weight indicators such as script count, stylesheet count, DOM size, iframe count, inline script/style indicators and long-task signals.
  • Page-type signals used to classify the page as a homepage, collection page, product page, landing page, blog article or standard page.
  • Locally generated Markdown exports when the user clicks Export Analysis.

What DeepGauge Does Not Collect

DeepGauge does not collect names, email addresses, payment information, health information, authentication credentials, personal communications, precise location, keystrokes, mouse movements or form submissions. It does not run in the background to build a browsing-history profile.

How Data Is Used

DeepGauge uses active-page scan data only to show the user a local optimization dashboard and, when requested by the user, to generate a local Markdown report download.

Data Sharing

DeepGauge does not transmit scan results, page content, website URLs, exported Markdown reports or settings to a developer-operated backend. Data is not sold or transferred to third parties.

Local Storage

DeepGauge may store the current target tab, popup-window state and last scan summary in Chrome extension storage on the user's device. This information remains local to the browser unless the user removes the extension or clears browser data.

Remote Code

DeepGauge does not load or execute remotely hosted code. All executable JavaScript is packaged with the extension.

Limitations

DeepGauge is a local optimization-audit tool. It does not require website admin access, analytics access, Shopify access or other account credentials. It does not bypass logins, paywalls, platform protections, browser restrictions or private website areas.

Contact

For support or privacy questions, use the support contact listed on the Chrome Web Store item page.
